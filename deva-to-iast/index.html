﻿ Devanāgarī to IAST converter for TEI files

Convert TEI XML files from Devanāgarī to IAST

Each XML file should have:

  1. A siglum (<idno type="siglum">[siglum]</idno> or <text corresp="#[siglum]">)
  2. For each paragraph or verse to be collated, an xml:id (e.g., <p xml:id="[id]">, <lg xml:id="[id]">
